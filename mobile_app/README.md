# Sprint 12 — Mobile Agent Flutter

Foundation app pou ajan yo.

## Screens
- Login
- Dashboard
- Nouvelle fiche
- Liste fiches
- Tirages
- Balance

## Configuration
Nan `lib/config.dart`, mete URL backend ou:

```dart
const String apiBaseUrl = 'https://your-domain.com/api/mobile';
```

Pou lokal:

```dart
const String apiBaseUrl = 'http://10.0.2.2/lotto_system/api/mobile';
```

## Run

```bash
flutter pub get
flutter run
```
