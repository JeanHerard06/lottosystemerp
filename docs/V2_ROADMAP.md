# Lotto ERP Enterprise v2.0 Roadmap

## Goal
Move from v1.0 RC1 into v2.0 enterprise platform development.

## Major Epics
1. API v2
2. Flutter Mobile Enterprise
3. Business Intelligence Dashboard
4. Fraud Detection Engine
5. Customer Portal
6. Plugin / Marketplace Architecture
7. AI Risk Assistant
8. Advanced SaaS Billing

## Branches
- `develop/v2.0`
- `feature/api-v2`
- `feature/flutter-mobile`
- `feature/bi-analytics`
- `feature/fraud-engine`
- `feature/customer-portal`
- `feature/plugin-marketplace`
- `feature/ai-risk-assistant`
- `feature/saas-billing-v2`

## Release Flow
1. Tag `v1.0-rc1`
2. Create branch `develop/v2.0`
3. Create issues from `docs/V2_ISSUES.md`
4. Work feature by feature
5. Merge via Pull Requests only
