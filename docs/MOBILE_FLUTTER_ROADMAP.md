# Mobile Flutter Roadmap

## Objectif
Créer une application agent mobile prête pour Android/iOS.

## Modules
- Login sécurisé
- Dashboard agent
- Vente ticket
- Ticket preview + impression Bluetooth
- Historique fiches
- Résultats tirages
- Balance/commission
- Notifications push
- Mode offline + sync
